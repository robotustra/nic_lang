To build:

	make
	
	./nic -c

If you modify code in words you must do:

	make clean

	make

	./nic -c


To run interpreter without recompiling words do just

	./nic


To see all list of words in the interpreter do:

	list

TO see what the [word] does:
	
	:word help

For instance type:

	:help help
